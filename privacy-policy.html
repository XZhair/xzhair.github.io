# Política de Privacidad de Smart X Companion Fecha de entrada en vigor: 14/03/2025 ## 1. Información que Recopilamos Smart X Companion no recopila, almacena ni transfiere datos de los usuarios de forma directa. La extensión accede al contenido visible de X (tweets, enlaces) en la pestaña activa para proporcionar sus funciones (sugerir respuestas, filtrar contenido, analizar enlaces y detectar tendencias). Este procesamiento ocurre localmente o a través de la API de OpenAI, que es responsabilidad del usuario configurar con su propia clave. ## 2. Uso de los Datos Los datos de X se usan exclusivamente para las funciones de la extensión y no se almacenan ni se comparten con terceros, excepto con OpenAI para procesar solicitudes de IA, según las políticas de OpenAI. ## 3. No Venta ni Transferencia No vendemos ni transferimos datos de usuarios a terceros, salvo que sea necesario para el funcionamiento de la extensión con la API de OpenAI. ## 4. Seguridad No recopilamos información personal ni autenticación. La seguridad depende de la clave API que el usuario proporciona. ## 5. Cambios en esta Política Cualquier cambio se notificará aquí. Consulta regularmente esta página. ## 6. Contacto Para preguntas, contacta a support@gobrayen.com.
